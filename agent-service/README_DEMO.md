1. **🎯 随机演示案例**

3. **💬 自定义输入**
   - 支持用户输入自定义问题
   - 实时流式响应
   - 智能意图识别

   - 现代化渐变设计

   - 动画效果

### 📊 演示流程

1. **用户点击演示按钮** → 随机选择该类别的问题
2. **AI管家分析** → 显示意图识别结果
3. **智能转接** → 显示转接动画
4. **专科回复** → 展示专业建议
5. **完成演示** → 显示成功状态

### 🎲 随机问题库

每个演示类别都包含5个不同的随机问题：

**🏥 健康紧急情况:**
- 呕吐、腹泻、精神不振
- 突然不吃东西、流口水
- 走路一瘸一拐、疑似骨折
- 呼吸急促、舌头发紫
- 眼睛红肿、流眼泪

**🍽️ 营养咨询:**
- 体重超标、减肥建议
- 掉毛严重、营养补充
- 幼犬喂养、喂食频率
- 食物过敏、避免清单
- 老年犬饮食注意事项

**🎾 训练咨询:**
- 基础服从训练（坐下）
- 乱叫问题纠正
- 咬东西行为纠正
- 随行训练
- 胆小性格改善

**❓ FAQ咨询:**
- 项圈清洗方法
- 充电相关问题
- 数据解读指导
- 定位找回功能
- 防水性能说明

**🎨 头像生成:**
- 赛博卡通风格
- 专业帅气风格
- Q版萌系风格
- 写实逼真风格
- 动漫可爱风格
